# Speech-Emotion-recognition-system-using-ML-IOT
This project builds a Speech Emotion Recognition (SER) system that detects eight emotions—Happy, Sad, Disgust, Surprise, Fear, Angry, Calm, and Neutral—from audio signals. Using MFCC feature extraction and a CNN–LSTM deep learning model, the system classifies emotions in real time and sends the results to an IoT platform for monitoring
